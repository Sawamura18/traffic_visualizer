# Traffic Visualizer
A Flask-based web app for visualizing highway traffic flow, density, and speed heatmaps using real sensor data.

## Features
- Upload raw traffic dataset (`.mat` or `.csv`)
- Generate space-time heatmaps for flow, speed, and density
- Download macro-level CSV summaries
- Built with Flask, Plotly, Pandas, and NumPy

## Run locally
```bash
pip install -r requirements.txt
python app.py
